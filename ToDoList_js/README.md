# JavaScript로 todo list 만들기

# 기능
- todo 추가
  - enter 키로 추가 가능

- todo 수정
  - 수정시, update 현재 일시로 변경

- todo 삭제

- todo 모두 삭제

- 완료
  - 완료시, 취소선
